A bit about me and this blog.

Welcome to Mirror – a space where I explore the intersection of software engineering, scientific computing, and thoughtful system design.

Who am I?

I'm a software engineer passionate about building elegant, maintainable systems. I believe in the power of well-crafted code and clear communication.

What is this blog about?

Here you'll find deep dives into:

  • Software architecture and design patterns
  • Web development with modern technologies

  • Performance optimization and best practices

The Tech Stack

This blog is built with:

  • Next.js 15 – React framework with App Router
  • Turborepo – Monorepo management
  • Tailwind CSS – Utility-first styling
  • MDX – Markdown with React components
  • Velite – Type-safe content layer
